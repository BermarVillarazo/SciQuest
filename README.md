# Project README

## Tech Stack

- **Roblox Studio:** v0.702.0.7020614 (64-bit)
- **Scripting Language:** Luau (Roblox’s built-in scripting language)
- **Data Management:** Data Stores Manager

## Deployment Instructions

### Install Roblox Studio
- Download and install Roblox Studio from the official Roblox website.

### Open Roblox Studio
- Launch the Roblox Studio application on your computer.

### Publish the Game
- Click **File** in the top-left corner.
- Select **Publish to Roblox…** or press **Ctrl + P**.
